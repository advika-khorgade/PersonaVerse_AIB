# Distribution Features - User Guide

## Quick Access

### From Main Navigation
Click **"Distribution"** tab in the top navigation bar to access:
- Distribution Tools (default view)
- Content Calendar

### From Content Editor
After generating content, you'll see new buttons:
- **Export** dropdown (top right)
- **Remix** button (top right)
- **Platform Formatting** section (below content)

---

## Feature 1: Download as Image 📸

### What It Does
Export your content as styled images for social media.

### How to Use
1. Generate content in Content Editor OR enter content in Distribution Tools
2. Click **"Export"** dropdown button
3. Choose format:
   - **Instagram Square** (1080x1080) - Perfect for Instagram posts
   - **Quote Card** (1200x630) - Great for Twitter/LinkedIn headers
   - **Carousel Slide** (1080x1080) - Multi-slide Instagram posts
4. Image downloads automatically as PNG

### Tips
- Images adapt to your current theme (light/dark)
- Content is automatically formatted for readability
- Persona name appears on the image

---

## Feature 2: Copy-to-Platform Mode 📋

### What It Does
Auto-format your content for different social media platforms with one click.

### How to Use
1. Generate or enter content
2. Scroll to **"Platform Formatting"** section
3. Click platform button:
   - **Copy for LinkedIn** - Adds paragraph spacing + hashtags
   - **Copy for Twitter** - Splits into numbered thread if needed
   - **Copy for Instagram** - Adds emoji spacing + hashtags
4. Content is copied to clipboard
5. Toast notification confirms copy
6. Preview shows formatted content

### Platform-Specific Features

**LinkedIn:**
```
Your content here...

With proper paragraph spacing...

#Hashtag1 #Hashtag2 #Hashtag3
```

**Twitter:**
```
1/3 First tweet with your opening...

2/3 Middle content continues...

3/3 Final thoughts and conclusion
```

**Instagram:**
```
✨ Your content here...

With emoji spacing...

.
.
.
#Hashtag1 #Hashtag2 #Hashtag3 #Hashtag4
```

## Feature 3: Content Calendar 📅

### What It Does
Visual calendar for scheduling and tracking content across platforms.

### How to Use
1. Go to **Distribution** tab
2. Click **"Content Calendar"** view
3. Navigate months using arrow buttons
4. Click any date to:
   - View scheduled content
   - Schedule new content
5. Status indicators show content state:
   - 🔵 **Planned** - Scheduled for future
   - 🟢 **Published** - Already live
   - ⚪ **Draft** - Work in progress

### Calendar Features
- Month view with all dates
- Visual status dots on dates with content
- Click date to see details
- Today's date highlighted
- Easy month navigation

## Feature 4: Content Remix 🎨

### What It Does
Transform your content into different formats using smart text processing.

### How to Use
1. Generate or enter content
2. Click **"Remix"** button
3. Choose format:
   - **Twitter Thread** - Numbered tweets with hook + CTA
   - **Carousel Slides** - Multiple slides with key points
   - **Short Hook** - Punchy attention-grabber
   - **LinkedIn Longform** - Professional post with hashtags
4. Edit remixed content in modal
5. Click **"Save Remixed Content"**
6. Content appears in editor

### Remix Formats Explained

**Twitter Thread:**
- Extracts hook from content
- Splits into 280-character tweets
- Numbers each tweet (1/n format)
- Adds call-to-action at end
- Perfect for long-form Twitter content

**Carousel Slides:**
- Creates title slide with hook
- Extracts key points (one per slide)
- Adds summary slide at end
- Shows slide separators
- Ready for Instagram/LinkedIn carousels

**Short Hook:**
- Creates punchy opening line
- Removes filler words
- Adds relevant emoji
- Perfect for attention-grabbing
- Use as post opener

**LinkedIn Longform:**
- Professional formatting
- Numbered key points
- Personal touch at end
- Relevant hashtags
- Optimized for LinkedIn algorithm

## Workflow Examples

### Example 1: Create Instagram Post
1. Generate content in Content Editor
2. Click **"Remix"** → Choose **"Short Hook"**
3. Save remixed hook
4. Click **"Copy for Instagram"** to format
5. Click **"Export"** → Choose **"Instagram Square"**
6. Post hook + image on Instagram

### Example 2: Create Twitter Thread
1. Enter long-form content
2. Click **"Remix"** → Choose **"Twitter Thread"**
3. Review numbered tweets
4. Edit if needed
5. Click **"Copy for Twitter"**
6. Paste thread on Twitter

### Example 3: Schedule Content
1. Create content in editor
2. Go to **Distribution** → **Calendar**
3. Click target date
4. Click **"Schedule Content"**
5. Content appears on calendar with status

### Example 4: Multi-Platform Distribution
1. Generate content
2. **Export** as Instagram Square
3. **Copy for LinkedIn** (formatted version)
4. **Remix** to Twitter Thread
5. **Schedule** all versions in calendar
6. Distribute across platforms

## Tips & Best Practices

### For Image Export
- Use **Quote Card** for professional quotes
- Use **Instagram Square** for social posts
- Use **Carousel** for multi-part content
- Switch theme before export for different looks

### For Platform Formatting
- LinkedIn: Best for professional, longer content
- Twitter: Best for quick thoughts, threads
- Instagram: Best for visual, emoji-rich content
- Always review preview before posting

### For Content Remix
- Start with **Short Hook** to test engagement
- Use **Twitter Thread** for detailed explanations
- Use **Carousel** for step-by-step guides
- Use **LinkedIn Longform** for thought leadership

### For Calendar
- Plan content 1-2 weeks ahead
- Use status colors to track progress
- Schedule different formats for same content
- Review calendar weekly

## Troubleshooting

### Image Export Not Working
- Check if content is entered
- Try different format
- Ensure browser allows downloads
- Check console for errors

### Copy Not Working
- Ensure content is entered
- Check clipboard permissions
- Try manual copy from preview
- Refresh page if needed

### Calendar Not Showing Content
- Backend integration needed for persistence
- Currently shows demo data
- Full functionality coming soon

### Remix Not Saving
- Ensure content is edited
- Check for modal close
- Try different format
- Refresh if needed
